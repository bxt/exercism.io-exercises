Haskell Trail
=============

Language Level
--------------

`**`

Workflow
--------
    
    # Prepare test:
    cat ../void.hs >> *_test.hs
    
    # Run test:
    runhaskell xxx_test.hs
    
    # REPL:
    ghci xxx.hs
    
    # Lint:
    pointfree 'xxx'
    hlint file.hs
